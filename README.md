## About This Site

Built with pure HTML, CSS, and JavaScript — no React, no Vue, no build tools. Just clean, fast, hand-written code.

**Features:**
- Animated hero with Matrix-style code rain canvas
- Scroll-reveal animations throughout
- Custom gold cursor with lag-follow effect
- 3D tilt effect on project cards
- Sticky experience tabs
- Animated skill bars
- Working contact form via Formspree
- Fully responsive — mobile, tablet, desktop
- WebP project screenshots for fast loading
- Favicon, OG tags, theme-color — production ready

---

## Tech Stack

```
Frontend   → HTML5, CSS3, JavaScript ES6+
Animations → CSS keyframes + IntersectionObserver
Form       → Formspree (AJAX, no backend needed)
Fonts      → Bebas Neue · Manrope · Fira Code (Google Fonts)
Images     → WebP format, lazy loaded
Hosting    → GitHub Pages / Netlify / Vercel
```
